Playground

Tulis HTML, CSS, dan JavaScript di sini — hasilnya muncul langsung di panel sebelah. Tidak perlu install apa-apa, tidak perlu simpan file. Coba preset yang sudah disediakan, atau mulai dari nol.

Contoh:
Preview auto-update

💡 Tips Penggunaan

  • Preview otomatis terupdate setiap kali kamu berhenti mengetik ~600ms. Kalau tidak sabar, klik tombol ▶ Jalankan.
  • Tekan Tab di editor untuk menambahkan 2 spasi (indentasi).
  • Coba preset di atas untuk melihat contoh, lalu modifikasi semaumu.
  • Kode tidak tersimpan saat tutup browser. Kalau mau disimpan, salin ke file .html di komputermu.
  • JavaScript console.log tampil di Console browser — tekan F12 untuk membukanya.
